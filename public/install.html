How to Install

1

Download the extension file below.

2

Open Chrome and go to chrome://extensions

3

Enable Developer Mode using the toggle in the top-right corner.

4

Drag and drop the downloaded .crx file onto the extensions page.

5

Click Add Extension when prompted.

⬇️ Download Yantra Extension

Compatible with Google Chrome and Chromium-based browsers.
Developer mode must be enabled to install.
